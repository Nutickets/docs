<Update label="22nd November 2023" description="Release R27">


**Access Control (android)**

* Continuous laser scanning on sunmi devices. Scanning next ticket is possible directly from ticket view without closing it (both in main entry and zones) [[NG-4611](https://linear.app/nuweb-group/issue/NG-4611)]
* Enhanced ticket scan result view. Added displaying email and order reference to main entry scan, added consistent headers and email to zone scan result. [[NG-4609](https://linear.app/nuweb-group/issue/NG-4609)]
* Add searching attendance lists by email and order reference (main entry and zones) [[NG-4610](https://linear.app/nuweb-group/issue/NG-4610)]


**Box office (android)**

* Added cash input without decimal point button which is more consistent with card terminal [[NG-4603](https://linear.app/nuweb-group/issue/NG-4603)]
* Hiding off sale and out of stock items when hiding is enabled in settings [[NG-4754](https://linear.app/nuweb-group/issue/NG-4754)]
* Fix displaying event/company fees [[NG-4601](https://linear.app/nuweb-group/issue/NG-4601)]


**All android apps**

* Add Portuguese to the list of supported languages [[NG-4641](https://linear.app/nuweb-group/issue/NG-4641)]

</Update>

<Update label="6th June 2023" description="Release R21">


**Box office**

* Added auto cutter functionality. After printing tickets and receipt paper can be automatically cut after each ticket or after the whole order (depending on the selected setting in the app) [[NG-3217](https://linear.app/nuweb-group/issue/NG-3217)]
*Available on devices that have paper cutter (tested on* `D2 mini`*; should also work on* `K2`*,* `K2 mini`*).*
* Auto printing of tickets in kiosk mode [[NG-3213](https://linear.app/nuweb-group/issue/NG-3213)]
*Available on* `Sunmi` *devices equipped with printer*
* Globally added header to requests indicating whether app is operating in kiosk mode, so that all endpoints can return data relevant for kiosk or normal mode [[NG-3252](https://linear.app/nuweb-group/issue/NG-3252)]
* Events that only have seated tickets are not shown in app [[NG-3211](https://linear.app/nuweb-group/issue/NG-3211)]
* Added company logo to kiosk welcome screen [[NG-3212](https://linear.app/nuweb-group/issue/NG-3212)]
* Added option to sell season tickets [[NG-3215](https://linear.app/nuweb-group/issue/NG-3215)]
* Fix for app making too many requests when cashless was half configured in admin panel (e.g. cashless was enabled but no shop was created) [[NG-2855](https://linear.app/nuweb-group/issue/NG-2855)]
