## Overview

System defaults are reusable templates for supported configuration types. They reduce repetitive setup and improve consistency.

## Capabilities

- Create, edit, delete, and restore defaults.
- Filter by type, name, created date, and trash state.
- Scope results by user visibility rules.

## Typical Use Cases

- Predefined content/schema defaults.
- Standardized setup for recurring event or configuration patterns.
