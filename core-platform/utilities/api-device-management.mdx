## Overview

API device management lists registered devices and allows operational controls such as blocking compromised devices.

## Supported Actions

- Search/filter devices.
- Rename device.
- Block/unblock by setting `blocked_at` state.
- Review last activity ordering.

## Typical Use Cases

- Disabling lost/stolen hardware.
- Temporarily blocking suspicious integrations.
- Cleaning up stale/unused devices.
