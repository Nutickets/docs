## Overview

Box office mode is launched from admin and redirected into signed shop routes.

## Supported Launch Inputs

| Parameter | Purpose |
|---|---|
| `event` | Opens box office directly on an event page |
| `customer` | Pre-selects a customer context |
| `custom_url` | Uses a specific company custom-url target where available |
| `params` | Additional query params for deep-link behaviour |

## Start and Cancel

- **Start**: creates a secure access URL and redirects staff into shop box-office mode.
- **Cancel**: exits box-office mode, clears session context, and returns staff to previous/admin flow.
