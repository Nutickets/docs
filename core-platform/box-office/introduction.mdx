## Overview

- Box office mode runs inside the shop stack but with staff-authenticated context.
- Staff can start box office sessions from admin and optionally deep-link to a specific event or customer.
- The mode supports assisted selling, payment shortcuts, and operational tools such as quick-sell actions.

## What Box Office Mode Adds

| Area | Capability |
|---|---|
| Session control | Start/stop secure box-office sessions |
| Assisted checkout | Set basket customer, apply staff discounts/vouchers, process payments |
| Event operations | Jump directly into event pages from admin launch links |
| Basket tools | Instant checkout and quick-sell patterns for repeat orders |

## Access Model

- Starting box office requires sell-item permissions.
- Box office behaviour can be globally limited by company feature/limitation settings.
