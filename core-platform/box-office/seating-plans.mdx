## Overview

- Box office mode uses the same event seating configuration as public shop flows.
- Staff can select seats/spaces, apply price-band mapped sale items, and complete assisted checkout.
- Seat availability and hold restrictions remain governed by event seating-plan rules.

## Key Dependencies

- Event has an attached and synced seating plan.
- Price bands are mapped to sale items.
- Seat status/hold rules allow the selected spaces to be sold.
