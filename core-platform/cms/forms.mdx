## Overview

- Forms support custom capture journeys with optional API integrations and recipient routing.
- Each form has metadata, success messaging/redirect options, and assigned questions.
- Form responses are reportable, and forms can be previewed.

## Core Capabilities

| Capability | Notes |
|---|---|
| Data capture assignment | Attach and reorder question fields |
| Integrations | Configure API integration settings per form |
| Delivery | Set recipient email targets for submissions |
| Publishing | Use preview URLs and response reporting |
