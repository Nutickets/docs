## Overview

- Theme and branding settings control shop-level visual presentation.
- Company branding and theme configuration are managed from separate but related admin screens.
- Theme schema controls layout, menus, banners, and many shop UI behaviours.

## Common Areas

| Area | Examples |
|---|---|
| Logos/assets | Default, dark/light variants, favicon, default imagery |
| Theme schema | Header/menu layout, colour and spacing options |
| Shop behaviour | Display options that influence rendered components |

## Announcement Support

Top menu announcement content is managed within theme schema and displayed across shop pages when enabled.
