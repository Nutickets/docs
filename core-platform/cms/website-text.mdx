## Overview

- Website text allows controlled overrides of translation keys used by the shop.
- Overrides can be applied globally or scoped to contexts such as event, schedule, or customer group.
- Original language-line values remain available for reference.

## Contexts

| Context | Use case |
|---|---|
| Global | Company-wide wording adjustments |
| Event | Event-specific copy |
| Schedule | Shared copy across scheduled events |
| Customer group | Segment-specific wording variations |
