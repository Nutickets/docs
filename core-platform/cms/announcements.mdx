## Overview

- Announcement messaging is implemented as a theme schema option for the top navigation area.
- When enabled, announcement content renders above the menu across shop pages.
- Announcement appearance is controlled with content, colour, and spacing settings.

## Configuration Points

| Setting | Purpose |
|---|---|
| Show menu announcement | Enables announcement rendering |
| Announcement content | Rich text/html announcement body |
| Announcement colour | Visual emphasis styling |
| Announcement padding | Vertical spacing control |
