## Overview

- SEO controls are available on core content entities (events, listings, articles, and company-level settings).
- Metadata drives search result snippets and social share previews.
- Shop-level indexability options can be controlled through settings where enabled.

## Common Fields

| Field | Purpose |
|---|---|
| Meta title | Search/share headline |
| Meta description | Search/share summary |
| Meta image | Social preview image |
