## Overview

- Many core entities support translated fields (for example names, descriptions, and metadata).
- Language-line overrides can be managed through website text workflows.
- Translation publishing workflows also exist in hub tooling.

## Where Translations Apply

- Event and sale-item labels/content
- Listing/article content fields
- Website text overrides for shop strings
