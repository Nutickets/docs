## Overview

- Audit logs capture entity and user activity for traceability.
- Logs are available in both admin and hub areas where authorized.
- Deleted audit records can be restored when permitted.

## Typical Uses

- Investigate who changed key configuration
- Review timeline around operational incidents
- Support internal compliance and accountability checks

## Access

Access is permission-controlled and scoped by user visibility/role.
