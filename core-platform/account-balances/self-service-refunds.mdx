## Overview

- Self-service balance refunds allow customers to move eligible order value into wallet balance without staff intervention.
- This behaviour is controlled by balance-refund settings at company level.
- Availability can be limited with a cutoff window and fee-refund toggles.

## Balance Refund Settings

| Setting | Behaviour |
|---|---|
| Enabled | Turns on balance-refund capability |
| Self serve enabled | Exposes customer-facing refund flow |
| Cutoff hours | Blocks late refund requests close to event start |
| Refund processing fees | Includes processing fees in wallet refund value |
| Refund reseller fees | Includes reseller fees in wallet refund value |

## Access Model

- Feature availability depends on account-balance refund feature context.
- Admin-only and admin+customer modes are both supported through feature context configuration.
