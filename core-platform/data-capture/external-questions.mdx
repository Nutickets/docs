## Overview

- External questions are customer-facing prompts shown in checkout/forms and related public interactions.
- They are created and managed from the main data-capture question library.
- Questions can be assigned/reordered per target context.

## Typical Uses

- Checkout profile and event-specific details
- Form submissions
- Optional marketing preferences
