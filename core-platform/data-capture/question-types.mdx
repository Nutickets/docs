## Overview

The platform supports multiple question types, including text, numeric, date/time, and option-based formats.

## Supported Types

| Type | System name |
|---|---|
| Short text | `text` |
| Long text | `long_text` |
| Selection | `select` |
| Number | `number` |
| Checkbox | `checkbox` |
| Multiple checkboxes | `multi_checkbox` |
| Radio buttons | `radio` |
| Yes / No | `yes_no` |
| Phone | `phone` |
| Email | `email` |
| Date | `date` |
| Date & time | `datetime` |
| Date range | `daterange` |
| Date & time range | `datetimerange` |

## Notes

- Some types support option lists.
- Some types are eligible for marketing-question use.
- Validation differs by type (for example email and phone validation paths).
